{{- /* registration-list Renders a sortable 3-column table of registrants from a Hugo data file. Usage: {{< registration-list data="barge_2026_registrations" >}} Supports two data-file shapes: 1. Live shape (current active event; written by zeffy_poll.py): last_updated: "2026-06-01 17:30 UTC" registrations: - full_name: Andy Bartalone ticket: 2 nickname: Buffalo - ... When this shape is detected, a "Last updated: ..." line is rendered above the table to signal that the automation is alive. 2. Legacy / past-event shape (e.g. ATLARGE 2026; hand-merged): - full_name: ... ticket: 1 nickname: ... - ... No "Last updated" line; the file is static. Columns: # — registration order (ticket number); numeric sort Name — Firstname + last-initial (e.g. "Andy B."); text sort Nickname — bolded; text sort Initial sort is by registration order ascending. Click any column header to re-sort by that column; click again to reverse. */ -}} {{- $key := .Get "data" -}} {{- $data := index hugo.Data $key -}} {{- $rows := $data -}} {{- $lastUpdated := "" -}} {{- if reflect.IsMap $data -}} {{- $rows = index $data "registrations" -}} {{- $lastUpdated = index $data "last_updated" -}} {{- end -}} {{- if not $rows -}}

No registrations to display.

{{- else -}} {{- with $lastUpdated }}

Last updated: {{ . }}

{{- end }} {{- range sort $rows "ticket" "asc" }} {{- $parts := split (.full_name | default "") " " -}} {{- $first := index $parts 0 -}} {{- $last := index $parts (sub (len $parts) 1) -}} {{- $lastInitial := substr $last 0 1 -}} {{- end }}
# Name Nickname
{{ .ticket }} {{ $first }} {{ $lastInitial }}. {{ with .nickname }}{{ . }}{{ end }}
{{- end -}}
