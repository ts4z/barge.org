On behalf of the Reformed BARGE Chip Committee, I'm pleased to announce we are producing chips for [2026](/chips/gallery/2026/), as well as [2024](/chips/gallery/2024) and [2022](/chips/gallery/2022/).


The [2026](/chips/gallery/2026/) chip theme, chosen by 2025 Main Event winner
Gillian Groves, is Whales. The chipset has eight chips, each featuring a
different whale and denomination. This set has 8 chips and costs $24.

Our [2024](/chips/gallery/2024/) theme theme is Omaholic’s 8 (or better) Steps
to Achieving Poker Wizardry, a theme picked by 2023 winner Rich Bremer. Each
chip features Vernon Donk, who accompanies Rich in poker tournaments.  This set
also has 8 chips and costs $24.

For [2022](/chips/gallery/2022/), our 2021 main event winner, the first after
the Covid year, is Sabyl “KaisMom” Landrum. Sabyl chose the theme “Mixed
Games”. Each chip depicts a different mix.  This set has 10 chips and costs
$30.

Save $8 when buying all three sets!

More information on each set is available on the gallery pages.

Each set also has a limited-edition purple chip with the main event winner's
picture. These are given to 1st, 2nd, and 3rd places in major events, and 1st
place winners in Second Chance events.

---

We are making new ["Welcome" chips](/chips/gallery/2026-welcome/), reviving
what BCC did this back in 2017.  The intent is that these chips can be given to
anyone you'd like to invite to BARGE. We have six of these available in six
colors and three different basic designs. These chips are $5 each or 6 chips,
$12. 18 chips, $24.  90 chips, $90.  Mix and match!

We also have some [2017 "Welcome" chips](/chips/gallery/2017-welcome/) for
sale.  These are different from the 2026 remakes, but equal in their
chippiness. These are $2 each.

Finally, we have a VERY limited number of chips for sale from [last

they were last year. But we can offer a $16 discount if you buy all four sets!
